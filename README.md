# 📱 Smart Calculator – Android App

A **Smart Calculator Android Application** built using Android programming.
This calculator performs basic arithmetic operations and provides a **clean user interface** with a **splash screen and calculator icon** for a better user experience.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖ Multiplication
* ➗ Division
* 🧮 Simple and user-friendly calculator interface
* 📱 Responsive Android UI
* 🌟 Splash Screen with Calculator Icon
* ⚡ Fast and lightweight application

---

## 🛠 Technologies Used

* **Java / Kotlin** (depending on what you used)
* **Android Studio**
* **XML Layout Design**
* **Android SDK**

---

## 📸 App Screens

### 1️⃣ Splash Screen

Displays a **calculator icon** when the app starts.

### 2️⃣ Calculator Screen

Main calculator interface where users can perform calculations.
